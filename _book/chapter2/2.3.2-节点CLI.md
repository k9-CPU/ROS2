# 2.3.2 节点常用CLI

CLI = Command Line Interface（命令行界面），就是那个黑乎乎的终端窗口，你敲命令的地方。

我们常用的节点CLI有哪些呢？

1、 运行节点（上文已经提到过，再加深一下印象）

```
ros2 run <packages_name> <executable_name>
```

2、 查看节点列表

```
ros2 node list
```

3、 查看节点信息

```
ros2 node info <node_name>
```
