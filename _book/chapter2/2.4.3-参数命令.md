# 2.4.3 操作参数的命令

1、查看节点的所有参数

```
ros2 param list
```

2、获取某个参数的值

```
ros2 param get <node_name> <param_name>
```

3、设置参数（立即生效）

```
ros2 param set <node_name> < param_name > <value>
```

4、把参数保存到文件

```
ros2 param dump <node_name>
```

5、启动时加载参数文件

```
ros2 run <packages_name> <node_name> --ros-args --params-file param.yaml
```

参数机制让节点变得灵活可配置，但普通节点的生命周期过于简单——启动即运行，关闭即销毁。
