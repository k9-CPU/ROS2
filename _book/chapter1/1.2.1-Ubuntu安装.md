# 1.2.1Ubuntu系统安装

1.首先是准备工作：

我们有一个安装了Windows11系统的电脑，准备一个8G以上的U盘（或者移动硬盘）

在安装Ubuntu之前要关闭独显直连。



2.查看/更改引导方式：

确保双系统安装完成后可以有双系统菜单：

Win+R呼出运行界面，输入：msinfo32，确定，会弹出一个系统信息的面板（或者直接搜索系统信息）。

![image-20260415164046385](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415164046385.png)

在项目栏找到BIOS模式，看对应的值，是否为UEFI，如果是就没问题

如果BIOS模式的值不是UEFI，而是显示传统（Legacy），那么就要改成UEFI，重启电脑按对应按键进入BIOS模式（这个不同的电脑用不同的按键，可以查一下自己电脑的BIOS进入方法）。

进入BIOS后开启UEFI，不同的电脑可能操作方法不同，下面的操作有对应的就按着修改没有就换一个：

- Boot，选择UEFI Boot回车选择Enabled
- Startup，Boot，UEFI/Legacy Boot旁边的选项选择修改为UEFI Only
- Boot mode select选择修改为UEFI
- Boot Type选择修改为UEFI Boot Type
- Launch CSM选择修改为Disabled。



3.查看/更改磁盘的分区格式

避免安装Ubuntu到最后却因为引导器安装而失败：

- 右键此电脑按钮选择管理，点击存储中的磁盘管理
- 在Windows所在的磁盘右键选择属性

![image-20260415165034025](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415165034025.png)

- 点击硬件再点击属性

![image-20260415165412211](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415165412211.png)

- 选择卷点击属性就可以看到磁盘分区形式了，是GPT就可以继续。

![image-20260415165401023](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415165401023.png)

如果磁盘分区形式显示的是：主启动记录(MBR)，那么就要修改为GPT格式：

- Win+R呼出运行界面，输入cmd确认打开命令行，输入命令：diskpart，回车，弹出这个界面：

![image-20260415165705833](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415165705833.png)

- 在这个界面输入命令：

```
list disk
```

- 选择要转换的Windows所在的磁盘

```
select disk=0
```

- 最后转换为GPT格式： 

```
convert gpt
```

-  确认一下：

```
list disk
```

- 如果显示出想要转化的磁盘在Gpt有了*标识那么就大功告成啦！然后接下来再去磁盘管理看就是GUID分区表(GPT)格式！

![image-20260415170228470](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415170228470.png)

4.关闭BitLocker加密

在设置-隐私和安全性里找到设备加密,把设备加密的开关关闭就好（磁盘已存储的数据越多，关闭设备加密耗时越长）。



5.压缩分区

在磁盘管理中挑选一个分区压缩给Ubuntu，右键选择压缩卷，可以先压缩200000MB，压缩后会发现多出来一个不到200GB的未分配分区。



6.制作Ubuntu安装盘

首先下载Ubuntu的映像文件，可以在[官网下载](https://ubuntu.com/#get-ubuntu)（真的很慢）也可以下载国内源的镜像文件，鱼鱼用的清华镜像站的，[点这个链接](https://launchpad.net/ubuntu/+mirror/mirrors.tuna.tsinghua.edu.cn-release)

![image-20260415170707438](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415170707438.png)

然后点进一个镜像站

![image-20260415170758844](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415170758844.png)

选择对应版本，我们选择22.04，选择这个desktop-amd64.iso版本

![image-20260415170938838](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415170938838.png)

下载过程中可以一起下载制作安装盘的工具，这里用的是ventoy，[这是ventoy下载链接](https://gitee.com/longpanda/Ventoy/releases/)，下载完成后记得解压再使用。

都下载完后，插上U盘，打开ventoy，选择你的U盘，在配置选项中选择分区类型，改为GPT,点击安装等待完成即可。接下来打开把下载好的Ubuntu映像文件粘贴到这个U盘里。

![image-20260415185331885](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415185331885.png)

![image-20260415185405615](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415185405615.png)

接着重启电脑（U盘不要拔下来哦）按下对应按键进入BIOS模式。

先关闭安全启动（Secure Boot），避免启动安装盘时出现Verification failed错误，

![image-20260415200426550](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415200426550.png)

接下来关闭Intel RST，避免Ubuntu安装过程出现Turn off RST的问题，找一找SATA Controller Mode这个磁盘的SATA管理器选项，选择设置为AHCI就可以了，有些电脑没有这个选项就没有问题不用管啦。

最后把U盘设置为最优先引导项，把要用的U盘设备按提示移到最前面。

![92a7070e82b095fe3479cc7212388442](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/92a7070e82b095fe3479cc7212388442.jpg)

**做完所有修改后记得保存再退出！！！**

然后重启就可以了，接下来就会U盘启动，出现这个Ventoy映像文件列表，选择Ubuntu22.04，启动(Boot in normal mode)，进入Ubuntu映像文件的引导菜单

![e84a1dfacd721f0931b662467570aab4](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/e84a1dfacd721f0931b662467570aab4.jpg)

选择Try or Install Ubuntu，就进入Ubuntu的安装程序了

![image-20260415201209120](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415201209120.png)

选择Try or Install Ubuntu后等待进入Ubuntu界面，弹出安装程序。

根据需要可以选择切换语言为简体中文，点击安装Ubuntu。

![image-20260415201841287](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415201841287.png)

默认正常安装。

选择继续，安装类型选择其他选项，选择继续，出现一个磁盘列表，找到设备为**空闲**且大小较我们之前磁盘分区大小稍大的那条，选中，点左下角的‘+’号，挂载点选择‘/’，点击‘OK’；再找到类型为efi分区的设备那条，看对应的设备号，在下面‘安装启动引导器的设备’下拉找到并选中（一般是Windows Boot Manager）。

![image-20260415202359402](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415202359402.png)

![image-20260415202529287](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415202529287.png)

最后点击现在安装，继续，选时区选择Shanghai，继续。

![image-20260415202658111](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415202658111.png)

接下来设置姓名，计算机名，用户名和密码。名字建议简短一点的英文，登录时是否需要密码按自己需要选择。继续。

等待安装完毕，出现安装完成的提示框后点击现在重启。重启时会看到‘Please remove the installation medium then press ENTER'，这时把U盘拔掉，然后敲回车。按回车电脑如果没反应直接按电源强制重启也可以。

![image-20260415202806046](./1.2.1-Ubuntu%E5%AE%89%E8%A3%85.assets/image-20260415202806046-1777186101394-28.png)
