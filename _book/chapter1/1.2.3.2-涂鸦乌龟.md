# 游戏二：涂鸦乌龟

游戏内容：启动海龟模拟器，启动海龟遥控器，控制海龟在地图上画出轨迹。



 1、启动海龟模拟器

```bash
ros2 run turtlesim turtlesim_node
```

![img](./1.2.3.2-%E6%B6%82%E9%B8%A6%E4%B9%8C%E9%BE%9F.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252FYwdkkkXZimWHCCOJXBzu%252Fimage.png)

出现这个小乌龟就说明成功了。



2、 启动海龟遥控器

```bash
ros2 run turtlesim turtle_teleop_key
```

![img](./1.2.3.2-%E6%B6%82%E9%B8%A6%E4%B9%8C%E9%BE%9F.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252FjnAfQis7ZbIUa8qSipVK%252Fimage.png)

出现这个提示之后用鼠标点击一下遥控器终端就可以用键盘遥控了。
