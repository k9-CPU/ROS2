# 游戏一：你说我听

1、 打开终端

运行发布者

```bash
ros2 topic pub /tcel std_msgs/msg/String "data: 'hello world'" -r 2
```

![img](../.gitbook/assets/spaces%2FJ2aPPrXUNE11MOXL9lWg%2Fuploads%2FfH8bk54FTFkyGP80DPzp%2Fimage.png) 出现这个，说明发布成功了。

运行订阅者

```bash
ros2 topic echo /tcel
```

![img](../.gitbook/assets/spaces%2FJ2aPPrXUNE11MOXL9lWg%2Fuploads%2Fl2m3QAOmJ2nGsyatQhgn%2Fimage.png) 出现这个，说明订阅成功了。
