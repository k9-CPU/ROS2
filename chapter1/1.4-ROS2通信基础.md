# 1.4ROS2通信基础

要理解ROS2如何高效运转，首先需要看清它的分层结构——就像剥开一颗洋葱，每一层都有不可替代的作用。

## 1.4.1ROS2架构图表

 

| 层级           | 组件                             | 作用                    |
| -------------- | -------------------------------- | ----------------------- |
| **应用层**     | 机器人算法、launch文件、用户节点 | 实现具体业务逻辑        |
| **客户端库层** | Rclcpp / rclpy                   | 封装DDS，提供ROS风格API |
| **抽象层**     | rmw（ROS Middleware Wrapper）    | 屏蔽不同DDS的差异       |
| **中间件层**   | DDS（Fast DDS/Cyclone DDS等）    | 发现、路由、QoS         |
| **操作系统层** | Linux/Windows/RTOS               | 底层资源管理            |

ROS架构竟然这么复杂，这都要记住吗，当然不是，只要知道每一层都有自己不可替代的活。
