1.2 动手安装ROS2

1.2动手安装ros2

本小节是 ROS2 的环境搭建实操模块，核心目标是完成 ROS2 在 Ubuntu 系统（以 22.04 Humble 版本为例）的完整安装与环境配置。

1.2.1Ubuntu系统安装

1.首先是准备工作

我们有一个安装了Windows11系统的电脑，准备一个8G以上的U盘（或者移动硬盘）

在安装Ubuntu之前要关闭独显直连。

2.查看/更改引导方式

确保双系统安装完成后可以有双系统菜单：

Win+R呼出运行界面，输入：msinfo32，确定，会弹出一个系统信息的面板（或者直接搜索系统信息）。

在项目栏找到BIOS模式，看对应的值，是否为UEFI，如果是就没问题

如果BIOS模式的值不是UEFI，而是显示传统（Legacy），那么就要改成UEFI，重启电脑按对应按键进入BIOS模式（这个不同的电脑用不同的按键，可以查一下自己电脑的BIOS进入方法）。

进入BIOS后开启UEFI，不同的电脑可能操作方法不同，下面的操作有对应的就按着修改没有就换一个：

  • Boot，选择UEFI Boot回车选择Enabled
  • Startup，Boot，UEFI/Legacy Boot旁边的选项选择修改为UEFI Only
  • Boot mode select选择修改为UEFI
  • Boot Type选择修改为UEFI Boot Type
  • Launch CSM选择修改为Disabled。

3.查看/更改磁盘的分区格式

避免安装Ubuntu到最后却因为引导器安装而失败：

  • 右键此电脑按钮选择管理，点击存储中的磁盘管理
  • 在Windows所在的磁盘右键选择属性

  • 点击硬件再点击属性

  • 选择卷点击属性就可以看到磁盘分区形式了，是GPT就可以继续。

如果磁盘分区形式显示的是：主启动记录(MBR)，那么就要修改为GPT格式：

  • Win+R呼出运行界面，输入cmd确认打开命令行，输入命令：diskpart，回车，弹出这个界面：

  • 在这个界面输入命令：
list disk
  • 选择要转换的Windows所在的磁盘
select disk=0
  • 最后转换为GPT格式： 
convert gpt
  • 确认一下：
list disk
  • 如果显示出想要转化的磁盘在Gpt有了*标识那么就大功告成啦！然后接下来再去磁盘管理看就是GUID分区表(GPT)格式！

4.关闭BitLocker加密

在设置-隐私和安全性里找到设备加密,把设备加密的开关关闭就好（磁盘已存储的数据越多，关闭设备加密耗时越长）。

5.压缩分区

在磁盘管理中挑选一个分区压缩给Ubuntu，右键选择压缩卷，可以先压缩200000MB，压缩后会发现多出来一个不到200GB的未分配分区。

6.制作Ubuntu安装盘

首先下载Ubuntu的映像文件，可以在官网下载（真的很慢）也可以下载国内源的镜像文件，鱼鱼用的清华镜像站的，点这个链接

然后点进一个镜像站

选择对应版本，我们选择22.04，选择这个desktop-amd64.iso版本

下载过程中可以一起下载制作安装盘的工具，这里用的是ventoy，这是ventoy下载链接，下载完成后记得解压再使用。

都下载完后，插上U盘，打开ventoy，选择你的U盘，在配置选项中选择分区类型，改为GPT,点击安装等待完成即可。接下来打开把下载好的Ubuntu映像文件粘贴到这个U盘里。

接着重启电脑（U盘不要拔下来哦）按下对应按键进入BIOS模式。

先关闭安全启动（Secure Boot），避免启动安装盘时出现Verification failed错误，

接下来关闭Intel RST，避免Ubuntu安装过程出现Turn off RST的问题，找一找SATA Controller Mode这个磁盘的SATA管理器选项，选择设置为AHCI就可以了，有些电脑没有这个选项就没有问题不用管啦。

最后把U盘设置为最优先引导项，把要用的U盘设备按提示移到最前面。

做完所有修改后记得保存再退出！！！

然后重启就可以了，接下来就会U盘启动，出现这个Ventoy映像文件列表，选择Ubuntu22.04，启动(Boot in normal mode)，进入Ubuntu映像文件的引导菜单

选择Try or Install Ubuntu，就进入Ubuntu的安装程序了

选择Try or Install Ubuntu后等待进入Ubuntu界面，弹出安装程序。

根据需要可以选择切换语言为简体中文，点击安装Ubuntu。

默认正常安装。

选择继续，安装类型选择其他选项，选择继续，出现一个磁盘列表，找到设备为空闲且大小较我们之前磁盘分区大小稍大的那条，选中，点左下角的‘+’号，挂载点选择‘/’，点击‘OK’；再找到类型为efi分区的设备那条，看对应的设备号，在下面‘安装启动引导器的设备’下拉找到并选中（一般是Windows Boot Manager）。

最后点击现在安装，继续，选时区选择Shanghai，继续。

接下来设置姓名，计算机名，用户名和密码。名字建议简短一点的英文，登录时是否需要密码按自己需要选择。继续。

等待安装完毕，出现安装完成的提示框后点击现在重启。重启时会看到‘Please remove the installation medium then press ENTER'，这时把U盘拔掉，然后敲回车。按回车电脑如果没反应直接按电源强制重启也可以。

1.2.2 一键安装 ROS2

这里感谢小鱼老师的支持

source <(wget -qO- http://fishros.com/install)

建议安装 humble 版本，稳的飞起，国内记得换源安装，如果你非要自己从头安装 ROS2，那我敬你是个勇者。

当出现这个界面时选 [1] 就是正常安装 ROS，如果对其他的安装感兴趣可以看看，小鱼老师的一键安装指令还是非常强大的。

上一步之后就会出现这个界面，然后选 [1]，这一步就是先自动更换系统源再安装 ROS。

这一步选 [2]，如果你之前装过别的系统源会自动清理，防止干扰。

出现这个界面时选 [1]，可以自动选择最快的源。

这一步选 [1] 就可以，也可以根据你的需要进行选择，比如：有时会选择官方源，有的包可能只在官方源有。

这一步选 [1]，我们使用 humble 版本的 ROS2。

这一步选 [1]，桌面版会自动把 rviz、rqt 等直接安装好。

当出现这个界面时就说明安装好了，输入你的密码就可以使用 ROS2 了。

1.2.3ROS2初体验

安装完成后，我们通过两个典型示例来验证环境是否正常，并直观理解话题通信机制。

游戏一：你说我听

1、 打开终端

运行发布者

ros2 topic pub /tcel std_msgs/msg/String "data: 'hello world'" -r 2

出现这个，说明发布成功了。

运行订阅者

ros2 topic echo /tcel

出现这个，说明订阅成功了。

游戏二：涂鸦乌龟

游戏内容：启动海龟模拟器，启动海龟遥控器，控制海龟在地图上画出轨迹。

1、启动海龟模拟器

ros2 run turtlesim turtlesim_node

出现这个小乌龟就说明成功了。

2、 启动海龟遥控器

ros2 run turtlesim turtle_teleop_key

出现这个提示之后用鼠标点击一下遥控器终端就可以用键盘遥控了。
