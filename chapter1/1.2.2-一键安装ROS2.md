# 1.2.2 一键安装 ROS2

这里感谢小鱼老师的支持

```bash
source <(wget -qO- http://fishros.com/install)
```

建议安装 humble 版本，稳的飞起，国内记得换源安装，如果你非要自己从头安装 ROS2，那我敬你是个勇者。

![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252F3dTBHyEaji5AmAY5KPG8%252Fimage.png)

当出现这个界面时选 [1] 就是正常安装 ROS，如果对其他的安装感兴趣可以看看，小鱼老师的一键安装指令还是非常强大的。



![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252FXBgmCXAhX67258S2hKqY%252Fimage.png)

上一步之后就会出现这个界面，然后选 [1]，这一步就是先自动更换系统源再安装 ROS。



![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252Fk0uhLdeGs66q8zhvbJ5z%252Fimage.png)

这一步选 [2]，如果你之前装过别的系统源会自动清理，防止干扰。



![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252FK1YOaj54XsWFNK9zrmZk%252Fimage.png)

出现这个界面时选 [1]，可以自动选择最快的源。



![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252FIpizwnuuWUg7bQdQEMUl%252Fimage.png)

这一步选 [1] 就可以，也可以根据你的需要进行选择，比如：有时会选择官方源，有的包可能只在官方源有。



![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252FzHSwCHZv2Tw88W25unzb%252Fimage.png)

这一步选 [1]，我们使用 humble 版本的 ROS2。



![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252FiUlb8fZPv2rDr15jCK68%252Fimage.png)

这一步选 [1]，桌面版会自动把 rviz、rqt 等直接安装好。



![img](./1.2.2-%E4%B8%80%E9%94%AE%E5%AE%89%E8%A3%85ROS2.assets/spaces%252FJ2aPPrXUNE11MOXL9lWg%252Fuploads%252F5WM14wVLLckxKnwZbr7O%252Fimage.png)

当出现这个界面时就说明安装好了，输入你的密码就可以使用 ROS2 了。
